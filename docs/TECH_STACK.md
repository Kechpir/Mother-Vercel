## UI/UX Концепция
- **Цвета**: Белый фон для контента, Темный (Zinc-900/Black) для акцентных блоков (Хедер, Форма, Футер).
- **Акцент**: Оранжевый (#ffa600) для кнопок, иконок и выделения текста.
- **Стиль**: Миниатюрность, глубокие тени, скругления (32px-48px), параллакс-эффекты.

## Payment (Prodamus)
- **Статус**: В планах интеграции.
- **Метод**: Редирект на платежную страницу + Webhook уведомление.

## Telegram
- **Группа**: Закрытая, с запретом копирования.
- **Бот**: Управление доступом и рассылка Zoom-ссылок.

## Zoom
- **Аккаунт**: Платная подписка ($50/мес).
- **Интеграция**: Публикация статичных ссылок в ТГ-группу (без API Zoom на текущем этапе).
