## Структура БД (Supabase)
### Таблица `participants`
- `id` (uuid, primary key)
- `full_name` (text, required) - ФИО
- `phone` (text, required) - Телефон
- `city` (text, required) - Город
- `email` (text, required) - Почта
- `status` (text, default: 'pending') - Статус (pending, paid)
- `payment_id` (text, nullable) - ID транзакции
- `created_at` (timestamp) - Дата записи
